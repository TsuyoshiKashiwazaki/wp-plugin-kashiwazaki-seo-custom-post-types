Kashiwazaki SEO Custom Post Types マニュアル

Kashiwazaki SEO Custom Post Types は、WordPressのカスタム投稿タイプをGUIで簡単に作成・管理するプラグインです。投稿タイプの作成・編集・削除、投稿タイプ間の記事移行、メニュー割り当て、カテゴリアイコン設定など、カスタム投稿タイプの運用に必要な機能をすべて管理画面から操作できます。

WordPressのカスタム投稿タイプは、コンテンツを論理的に分類しURL構造を最適化するための基盤です。本プラグインはGUIでの投稿タイプ作成・管理に加え、投稿タイプ間の記事移行やメニュー割り当て管理を提供します。

メニュー管理

カスタム投稿タイプごとのメニュー割り当てとカテゴリアイコン設定

主な特徴

  • カスタム投稿タイプのGUI管理 - コードを書かずにカスタム投稿タイプを作成・編集・削除できます
  • 親子階層管理 - 投稿タイプの親子関係を設定し、リライトルールを自動管理します
  • 投稿タイプ間の記事移行 - スラッグとタクソノミーを維持したまま、投稿タイプ間で記事を一括移行できます
  • メニュー割り当て管理 - カスタム投稿タイプごとにメニューの割り当てを管理できます
  • カテゴリアイコン設定 - カテゴリごとにアイコンを設定し、管理画面での視認性を向上させます
  • パーマリンク自動管理 - 投稿タイプの作成・変更時にリライトルールを自動で更新し、404エラーを防止します
  • 20種類のAJAXハンドラー - すべての操作を非同期で処理し、スムーズな管理体験を提供します

SEOとGEOへの効果

本プラグインは管理画面専用のツールですが、サイト構造の最適化を通じて検索エンジン最適化（SEO）と生成エンジン最適化（GEO）に貢献します。

  • 適切なURL構造の構築 - カスタム投稿タイプによる論理的なURL構造は、検索エンジンのクローラビリティを向上させます
  • 階層型投稿タイプの活用 - 親子関係を持つ投稿タイプにより、サイト全体の論理的なアーキテクチャを構築できます
  • リライトルールの自動管理 - サイト構造変更時のリライトルール管理により、リストラクチャリング中の404エラーを防止します
  • AI対応のコンテンツ整理 - 論理的なコンテンツ分類は、AIがサイトのトピックを正確に理解する助けになります

技術仕様

  • データ保存: wp_options (kstb_categories_migrated 等)
  • カスタムテーブル: なし
  • 外部通信: なし
  • Cron: なし
  • メール: なし
  • セキュリティ: nonce検証、manage_options権限、sanitize_text_field

動作要件

項目要件
WordPress5.0 以上
PHP7.0 以上
外部依存なし（外部HTTP通信・Cron・メール送信いずれも不使用）
